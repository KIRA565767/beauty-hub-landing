# Beauty Hub Docs / Source of Truth

Эта папка содержит финальные lock-документы проекта. Они важнее старых промтов, черновиков и визуальных экспериментов.

## Приоритет документов

1. `VISUAL_LOCK.md` — финальная визуальная система Beauty Hub. Если любой старый документ возвращает бетон, terminal, dashboard, SaaS, cyber/neon или random 3D — игнорировать старый документ.
2. `SITE_ARCHITECTURE_LOCK.md` — финальная архитектура: геометрия, scroll, route rooms, кнопки, mobile, переходы.

## Рабочий принцип

Не переизобретать концепт. Любая новая правка в коде должна проверяться против этих двух файлов.

## Короткая формула

Beauty Hub = dark polished editorial route archive / private beauty club для beauty-мастеров.

Не dashboard. Не terminal. Не SaaS. Не бетонный brutalism. Не обычный beauty landing.
