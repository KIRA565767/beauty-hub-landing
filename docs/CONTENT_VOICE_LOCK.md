## 0. Purpose

This document fixes the voice of Beauty Hub.

Visual design gives status. Text must sell the system.

If the visual layer looks premium but the copy sounds like generic beauty infobusiness, the whole project becomes cheaper.

Beauty Hub does not persuade through pressure. It diagnoses and shows a route.

Use this document before writing:

- website headlines;
- room copy;
- CTA labels;
- Telegram bridge text;
- button text;
- interface labels;
- pinned Telegram copy;
- product descriptions;
- future bot scripts.

---

## 1. Core voice principle

```text
The site does not motivate.
The site diagnoses.
The site does not shout.
The site gives a route.
```

Beauty Hub speaks to beauty masters as professionals.

Not as “girls”.
Not as helpless beginners.
Not as people who need hype.

The voice is strict, practical, and precise.

---

## 2. Tone of Voice

### 2.1 Quiet Authority

Julia speaks from 14 years of hands-on practice.

The tone is calm, senior, and exact.

It should feel like a professional who has already seen the repeated mistakes of the market and can identify the real node without drama.

Allowed tone:

```text
steady
experienced
clear
strict
practical
observant
calm
```

Forbidden tone:

```text
hysterical
motivational
begging
hype-driven
toxic
shaming
cheap luxury
fake scarcity
```

### 2.2 Surgical Precision

Do not name problems vaguely.

Wrong:

```text
You have no money.
You need to grow.
You need to level up.
```

Correct:

```text
The check is stuck.
The client does not reach booking.
The profile creates interest but not trust.
The service result is not stable enough to support a higher price.
```

Every problem should be named as a working node:

```text
technique
client path
price
schedule
booking
repeat visits
salon dependence
personal route
```

### 2.3 Editorial Restraint

The site must sound like an editorial archive, not a sales webinar.

Use short, controlled statements.

Avoid too much emotional pressure.

The reader should feel:

```text
This person understands my daily work.
This is not another motivational course.
There is a system here.
```

## 3. Forbidden vocabulary

Do not use the following words and constructions unless explicitly approved.

### 3.1 Audience labels

Forbidden:

```text
devочки
красотки
мастерицы
мои хорошие

```

Preferred:


Use “ты” calmly. No fake closeness.

Forbidden:

```text

взлом алгоритмов
закроешь 8 из 10
наставничество до результата
```

### 3.3 Shame and aggression

```text
если не сделаешь — останешься в болоте
```

Beauty Hub can be strict, but not toxic.


## 5. CTA rules

CTA should act like a strict access pass.

A button must not beg for a click.

It should clearly name the next action.

### 5.1 Global CTAs

Allowed:

```text
ВОЙТИ В TELEGRAM →
ПЕРЕЙТИ В ХОЛЛ МАРШРУТОВ →
ОТКРЫТЬ МАРШРУТЫ →
ПРОДОЛЖИТЬ МАРШРУТ →
```

Avoid:

```text
Подписаться
Залететь в канал
Купить сейчас
Успеть забрать
Хочу результат
```

### 5.2 Room-specific CTAs

Technique:

```text
ОТКРЫТЬ КУРС ПО ТЕХНИКЕ →
ПРОДОЛЖИТЬ МАРШРУТ ПО ТЕХНИКЕ →
```

Client Flow:

```text
РАЗОБРАТЬ КЛИЕНТСКИЙ ПОТОК В TELEGRAM →
РАЗОБРАТЬ ПУТЬ КЛИЕНТА ДО ЗАПИСИ →
```

Independence:

```text
СОБРАТЬ ПЛАН ВЫХОДА →
ПОЛУЧИТЬ ПЕРВЫЙ ПЛАН ВЫХОДА →
```

Mentorship:

```text
ЗАПРОСИТЬ ЛИЧНЫЙ РАЗБОР →
ПОНЯТЬ, С КАКОГО УЗЛА НАЧАТЬ →
```

### 5.3 Navigation microcopy

Use:

```text
← В ХОЛЛ
СЛЕДУЮЩАЯ КОМНАТА →
ВСЕ МАРШРУТЫ
СМЕЖНЫЙ МАРШРУТ
```

Avoid:

```text
Назад
Читать ещё
Подробнее
Следующий блок
```

## 7. Headline rules

### 7.1 Good headline structure

Use concrete tension:

```text
Рост ломается не там, где ты обычно пытаешься чинить.
```

```text
Если техника нестабильна, маркетинг только быстрее покажет слабое место.
```

```text
Клиентский поток ломается между интересом и записью.
```

```text
Выход из найма — это не мотивация, а расчёт.
```

```text
Когда проблема не одна, нужен не ещё один курс, а разбор узла.
```

Do not use:

```text
Хочешь больше клиентов?

Стань мастером мечты.
Зарабатывай больше уже завтра.

```

## 9. Telegram copy rules

Telegram copy must never sound like:

```text
подписывайся на канал
там много полезного
буду рада видеть
```

Correct Telegram framing:

```text
продолжить маршрут
получить первый материал
открыть разбор
получить доступ к живому пространству
```

### 9.1 Homepage Telegram copy

```text
Telegram — не канал с постами. Это живое пространство Beauty Hub, где маршруты продолжаются через материалы, разборы и первые шаги.
```

### 9.2 Technique Telegram bridge

```text
Продолжить маршрут по технике: первый материал — где чаще всего рождаются отслойки, потеря скорости и нестабильный результат.
```

### 9.3 Client Flow Telegram bridge

```text
Продолжить маршрут по клиентскому потоку: разобрать, где ломается связка профиль → фото → оффер → переписка → запись.
```

### 9.4 Independence Telegram bridge

```text
Продолжить маршрут самостоятельности: проверить, что должно быть готово до выхода из найма — запись, цена, график и запас.
```

### 9.5 Mentorship Telegram bridge

```text
Запросить личный разбор: передать ситуацию на диагностику, если проблема не укладывается в один маршрут.
```

## 11. Case copy rules

Cases must sound like case notes, not testimonials.

Correct structure:

```text
CASE 01 / ROUTE
Starting blind zone
Surgical correction
System result
```

Forbidden:

```text
Она была в шоке.
Она не верила, что так можно.
Результат превзошёл ожидания.
Минус страх, плюс деньги.
```

Allowed:

```text
Стартовая слепая зона: профиль собирал просмотры, но не переводил в запись.
Исправление: переписана логика первого экрана и ответов в переписке.
Системный результат: путь клиента стал короче и понятнее.
```

Use only real data.
Do not invent numbers.
Do not fake names.

## 12. Copy acceptance checklist

Before approving any page copy, check:

```text
Does it diagnose instead of motivate?
Does it name a concrete node?
Does it avoid infobusiness tone?
Does it avoid fake urgency?
Does it speak to a professional master?
Does it connect to the correct route?
Does the CTA name the next step?
Does Telegram feel like continuation, not subscription?
```

If any answer is no, rewrite.
